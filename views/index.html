Exercise tracker

Made by Liam Washburn for Free Code Camp's APIs and Microservices certification.

Create a New User

POST /api/exercise/new-user


Add exercises

POST /api/exercise/add

To view a users exercise log: http://fortunate-jump.glitch.me/api/exercise/log?{userId}[&from][&to][&limit]

Example: "http://fortunate-jump.glitch.me/api/exercise/log?userId=_RrRTmZg&from=1988-01-01&to=2018-07-23&limit=10"

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Made with JavaScript, HTML, NodeJS and MongoDB
